# Secure Online Voting System

This project is a Django-based secure online voting prototype developed as part of a final year Computer Science project. 

The system demonstrates:

- User authentication
- One-vote-per-user enforcement
- CSRF protection
- Secure database interaction using Django ORM
- Real-time vote counting
- Admin management of candidates and users

The goal of this prototype is to explore how web-based systems can reduce vulnerabilities present in traditional manual voting processes.
